Built with the help of **Claude Sonnet** (via Cursor) — showcasing proficiency with AI-powered development tools and modern coding workflows.

## Live Site

**[https://abhishekjani123.github.io/](https://abhishekjani123.github.io/)**

## Local Development

```bash
# Serve locally (port 3000)
npm run serve

# Verify before push
npm run verify
```

## Deployment (GitHub Pages)

The site deploys automatically when pushed to `main`. Ensure:

- `.nojekyll` is present (disables Jekyll processing)
- All assets (images, PDF) are committed
- Run `npm run verify` before pushing
